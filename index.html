MiContact

An app I designed to help people with Autism learn to make eye contact in a way that's comfortable for them. The first six levels were programmed using Swift with help from a Friend. With their permission, I reverse engineered the source code they provided me to learn how to code, and then I proceeded to the other 54 levels, which contain 4 images each.

QuizBot

This is a very basic python project I built last semester for a programming assignment. It's effectively a chat bot that provides the user with clues regarding a word that the user needs to guess. Each word has 5 clues. The user inputs their guess using the key board, but there are also key words that the user can enter to change dictionaries to guess a different set of words, or to change the scoring criteria for the bot. The bot relies on 4 different files to function. You can view more detailed information surrounding the code itself in the comments of each file. The files are available to view/download from GitHub at the link below.

Country Locator and Webcam Tool

This was a basic Processing sketch that was completed as a lab assignment for my Authoring Module last semester. The idea is that the user is presented with a map, and when they click on the map, the name of the country they've clicked on is returned, and the closest available webcam will be displayed in front of the map. To close the map, the user presses any key.

Star Wars Character Table

This is a Swift application I developed for my mobile multimedia class. It allows the user to view information about various characters from the star wars franchise. It includes elements from UIKit, WebKit and I will be expanding it to make use of Core Data so users can create a list of favourites

Paddle and Ball Game

This project uses pygame to create a simple paddle and ball game. The paddle can be moved left or right using the arrows on the keyboard, and the ball hits off of the sides of the screen. This is a smaller segment of a bigger project I'm currently working on for a programming assignment.
