Character Information App

An app I built using Python and Kivy, which displays a selection of characters from the video game "Hello Kitty Island Adventure". When the user clicks one of the character's names, a pop-up window appears displaying more information about the selected character.

QuizBot

This is a flashcard game I made with support from a 100 days of python course on Udemy. Flashcards are generated using data from a csv file. Each flashcard displays a word in French, and its English translation. The user can move to the next flashcard by pressing either the check mark or x buttons at the bottom of the screen.

Miles to Kilometers Tool

I created this Miles to Kilometers Converter Tool using tutorials from a 100 days of Python course on Udemy. The user can type a number of miles into the text file, and when they press the convert button, the equivalent number of kilometers is displayed.

Password Manager Tool

Using guidance from a 100 days of Python course on Udemy, I built this Password Manager Tool. It isn't fully secure, it's more of an example of the concept. Users can input their username and password, as well as the website they're for, and the app will save it to a csv file. The tool can also generate a secure password for the user.

Pomodoro Timer

This project was also a part of the 100 days of Python course on Udemy. The user can input the length they want a timer to run for, and the tool will countdown until the timer finishes.
